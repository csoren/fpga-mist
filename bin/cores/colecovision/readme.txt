A colecovision core for the MiST
---------------------------------------------------------------------------------

Buttons:
- Keyboard 0-9 	    -> 0-9
- Keyboard Z        -> fire/jump (Y on German Keyboard)
- Arrow Keys        -> Move
- Keyboard F12      -> OSD to select roms from the sd card (needs extension .col)

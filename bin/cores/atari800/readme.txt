Atari800 core for MiST by Mark Watson

Find the source code at: 
http://www.scrameta.net/autobuild/20140619/mist/build_NTSC_VGA/out/

Detailed instructions at:
http://www.scrameta.net/autobuild/20140619/instructions.txt

Quickstart:
- Use the latest firmware
- Rename atari800_XXX.rbf to core.rbf and put it into your SD cards root.
- Place atari800 roms atarixl.rom and ataribas.rom under atari800/rom
  to your SD card
- Optionally add some atr disk images under atari800/user ro your
  SD card
- Insert SD card, boot MiST
- Press F11 and F12 for menus
